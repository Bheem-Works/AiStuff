Miso Highlighter — Demo

This page is a simple demo page for the Miso Highlighter extension. Select text and right-click or press M to open the tools popup.

Try this:

  • Select a portion of the paragraph below and press M to highlight it.
  • Right-click a selection to open the full tools popup; use Save to copy text to clipboard.
  • Hover an existing highlight to see the minimal popup with Change Color and Save.
